DESCRIPTION

r.internal is a GRASS-GIS module that links inside a Mapset raster maps from other Mapsets. Linking can be either hard or soft. The idea is to avoid data duplication whenever it makes sense to do so. For example, in massive processing setups, where saving space might be critical.
